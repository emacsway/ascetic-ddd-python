# ascetic-ddd-python

Toolkit and [seedwork](https://martinfowler.com/bliki/Seedwork.html) for a Python DDD project.

Данный проект представляет собой компиляцию новых и файлов из различных моих Open Source библиотек,
реиспользуемых файлов и seedworks,
подсмотренных мною решений в различных Open Source Reference Applications,
транспайлинг на Python моих файлов с Golang Reference Application [Grade](https://github.com/emacsway/grade),
которые я наиболее часто использую в различных проектах.

В проекте используется принцип "Program into Your Language, Not in It".
Богатство языковых конструкций Python намеренно не используется по причине того,
что исходный код должен легко транспайлиться на другие языки программирования
(в частности, [ascetic-ddd-go](https://github.com/krew-solutions/ascetic-ddd-go)),
у которых языковое богатство поскромнее.

О структуре проекта и размещении тестовых файлов:

1. Из проекта в будущем могут быть выделены самостоятельные библиотеки.\
   Этим и продиктована его структура (много пакетов в корне).
2. Проект представляет собой, прежде всего, [seedwork](https://martinfowler.com/bliki/Seedwork.html), и его структура сделана таким образом,\
   чтобы было удобно копировать только релевантные его пакеты вместе с тестами.
3. Мне близка идеология Golang по поводу размещения тестового файла рядом с логикой. Пытался найти компромисс.

Внимание!
Проект пока еще в активной разработке, не production-ready, не прошел отладку и оптимизацию.
Структура проекта не стабильна и обратная совместимость до версии 1.0.0 не поддерживается.
